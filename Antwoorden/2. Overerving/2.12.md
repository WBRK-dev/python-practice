# Opdracht 2.12

## Het goede antwoord is: A

## Uitleg

MRO van S(Q, R): S Ôćĺ Q Ôćĺ R Ôćĺ P.  
`S.m()` roept via `super()` de chain aan:

- `S.m()` Ôćĺ `super().m()` = `Q.m()`
- `Q.m()` Ôćĺ `super().m() + 1` = `R.m() + 1`
- `R.m()` Ôćĺ `super().m() + 2` = `P.m() + 2`
- `P.m()` Ôćĺ `1`

Terug: `R.m() = 3`, `Q.m() = 4`, `S.m() = 4`.  
Resultaat: `4` (antwoord A).

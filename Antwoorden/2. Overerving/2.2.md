# Opdracht 2.2

## Het goede antwoord is: A

## Uitleg

`Beagle().sound()` roept `Beagle.sound()` aan.  
Deze roept via `super().sound()` de methode van `Dog` aan, die op zijn beurt via `super().sound()` de methode van `Animal` aanroept.  
De chain is: `Animal.sound()` geeft `"..."` → `Dog.sound()` geeft `"...woof"` → `Beagle.sound()` geeft `"...woof howl"`.

Resultaat: `"...woof howl"` (antwoord A).

## Uitleg

Wanneer de loop bij `"start"` komt, wordt `hasStarted` op `True` gezet.  
Vervolgens wordt **direct daarna** de check `if hasStarted:` uitgevoerd ‚Äî en op dat moment is `item` nog steeds `"start"`.  
De code probeert `int("start")` uit te voeren, wat een `ValueError` veroorzaakt omdat `"start"` niet naar een integer omgezet kan worden.  
Daarom gooit de applicatie een error (antwoord A).

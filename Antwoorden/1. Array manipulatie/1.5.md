# Opdracht 1.5

## Het goede antwoord is: C

## Uitleg

`hasStarted` begint als `True`. Zolang `hasStarted` `True` is (en we niet bij `"end"` zijn), worden letters opgeteld.  
Na `"end"` wordt `hasStarted` `False` en slaat `continue` de rest over.

- `"a"` → +1 (result=1)
- `"start"` → zet `hasStarted` op `True` (was al `True`), geen letter-match, geen optelling
- `"b"` → +2 (result=3)
- `"c"` → +3 (result=6)
- `"d"` → +4 (result=10)
- `"e"` → +5 (result=15)
- `"end"` → `hasStarted = False`, `continue`
- `"f"` → overgeslagen

`result = 15` (antwoord C).
